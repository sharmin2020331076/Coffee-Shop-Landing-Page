# Coffee Shop Landing Page ☕

A modern and responsive **coffee** shop landing page built with React, TypeScript, and Vite. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)
It showcases a fictional coffee brand with sections like hero, menu, about, and contact, designed as a front-end UI project. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)

## Features

- Responsive layout for desktop and mobile screens. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)
- Built with React + TypeScript + Vite for fast development and HMR. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)
- Modular component structure under `src` for easy customization. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)
- CSS styling for typography, layout, and coffee-themed visuals. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)

## Tech Stack

- React  
- TypeScript  
- Vite  
- CSS  
- ESLint (with TypeScript setup) [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)

## Getting Started

Clone the repository:

```bash
git clone https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page.git
cd Coffee-Shop-Landing-Page
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The app will be available at the local URL shown in your terminal (usually `http://localhost:5173`). [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)

## Available Scripts

- `npm run dev` – Start the Vite dev server with HMR. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)
- `npm run build` – Create a production build in the `dist` folder. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)
- `npm run preview` – Preview the production build locally. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)
- `npm run lint` – Run ESLint checks. [github](https://github.com/sharmin2020331076/Coffee-Shop-Landing-Page)
